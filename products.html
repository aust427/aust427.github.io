A selection of various programming projects that I have developed and contributed to. These projects cover 3D visualization, algorithm application, data science, and front-end development.

An analysis of my personal Spotify data as an attempt to create a song prediction algorithm using data analysis and machine learning. Barnes-Hut Algorithm

An application built for applying the Barnes-Hut algorithm to an n-body simulation. Built in JavaScript using Three.js. See the here.

Subhalo Particle Visualizer

An application built for visualizing and plotting subhalo particle data from the Illustris and IllustrisTNG simulations. Utilizes a JavaScript front-end and a Python back-end. The main libaries are Three.js, jQuery.js, and D3.js. See the Github repository here. Is currently in-development.

Astrosims

An online database used for hosting astronomical simulation data. Developed alongside Dylan Simon, PhD, and Elizabeth Lovero. Primary contributions include front end development, data visualization, and database management. See the website here.

Utah Teapot

An interactive rendering of the Utah Teapot. Developed using JavaScript and the WebGL library. Includes a toggle between environment mapping and basic shading, as well as object and world rotations. See the interactive example here.
