Data Releases

Description of data that I have contributed toward creating as part of my research. For access, please contact me, Dr. Rachel Somerville, or Dr. Shy Genel.

IllustrisTNG Rockstar & Consistent-Trees Catalogs

Catalogs produced by running the Rockstar and Consistent-Trees algorithms on IllustrisTNG volumes at all snapshots. Rockstar outputs available as either raw or Subfind-esque outputs. Subfind-esque outputs are compatable with the illustris_python module.

IllustrisTNG Subfind-Rockstar SubLink Tables

Linking tables containing bijective matches for dark matter halos identified by Rockstar and Subfind in IllustrisTNG, created by running the SubLink algorithm. Matches have been created between Rockstar dark matter-only (DMO) and Subfind DMO, and Rockstar DMO and Subfind full physics (FP). Matches exist at each snapshot for all resolutions.

IllustrisTNG Semi-Analytic Model Runs

Created by running the Santa-Cruz Semi-Analytic Model (SC-SAM) on IllustrisTNG Consistent-Tree merger trees. Available only at TNG100-1, TNG300-1, and TNG50-1 resolutions.
